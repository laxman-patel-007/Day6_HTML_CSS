HTML Basics

<html> — Root element that wraps entire HTML document. <head> — Contains metadata like title, links to CSS, charset. <body> — Visible part of HTML page. <h1> to <h6> — Headings, h1 is largest. <p> — Paragraph text. <div> — Generic container block. <span> — Inline container. <a href> — Hyperlink. <ul>, <ol>, <li> — Unordered, ordered lists and items. <table> — Table structure with <tr>, <th>, <td>.

CSS Basics

color: Changes text color. background-color: Sets background. padding: Space inside the element. margin: Space outside the element. border: Adds outline around the element. font-size: Controls text size. text-align: Aligns text. display: Controls layout (block, inline, flex). flexbox: Layout system for flexible items. box-shadow: Adds shadow around elements.
